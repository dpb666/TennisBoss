# USER.md

## User Profile
The user is the primary operator of the TennisBoss system.
They are responsible for strategy direction, validation of outputs, and system tuning.

## Role in System
- Final human decision maker
- Defines betting strategy and risk level
- Requests analyses, improvements, and automation
- Oversees AI predictions and adjusts system behavior

## Preferences
- Focus on tennis prediction and value betting
- Prefers clear, actionable outputs over long explanations
- Interested in automation, bots, and performance optimization
- Works with local AI models (Ollama / DeepSeek)

## Working Style
- Iterative improvement approach
- Tests strategies in real conditions
- Uses feedback loops from wins/losses
- Prefers practical results over theory

## System Interaction
- Communicates with OpenClaw agents (main, stats, odds, coder)
- Requests audits, analysis, and strategy generation
- Can modify system architecture and prompts

## Risk Handling
- Accepts calculated risk in betting strategies
- Focus on long-term value, not single outcomes
- Uses data-driven decisions whenever possible

## Objective
Build a high-performance tennis prediction system capable of identifying consistent value betting opportunities.
